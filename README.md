# rtx3080-scraper

1. set env variable
    * create a `.env` file
    * set EMAIL, PASSWORD, USER_EMAIL

2. Go to : https://www.google.com/settings/security/lesssecureapps  and set the Access for less secure apps setting to Enable 

3. start service

```
npm run start
```
